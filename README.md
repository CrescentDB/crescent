# crescent
## A Search Engine and Study Ecosystem for Students
**Version 1.0, 2025**

---

## Overview

crescent is a **privacy-focused, open-source search engine** designed specifically for students, researchers, and lifelong learners. It is more than just a search tool: crescent is a fully integrated learning ecosystem. With API access to Wikipedia, WolframAlpha, OpenLibrary, and other educational sources, it provides a seamless way to research, take notes, organize study material, and explore knowledge in a safe, ad-free environment.

The project is fully **open-source**, meaning you can examine, modify, and extend it under the **CrescentDB Supreme Protection License (CSPL) v3.3)**. crescent is designed to help students stay productive, organized, and focused without exposing personal data or relying on third-party trackers.

---

## Core Philosophy

1. **Student-Centric**: All features are optimized for learning, homework, research, and project management.  
2. **Privacy First**: crescent does **not collect any user data**, ensuring safe use for students of any age.  
3. **Open-Source Freedom**: Anyone can study the code, modify it, or contribute improvements under the CSPL license.  
4. **Seamless Integration**: Combines search, notes, and study tools in a single interface to minimize context switching.  
5. **Accessible and Safe**: No ads, no tracking, and a simple interface designed for learners under 18.

## Key Features

### Search & Research

- **Multi-Source Search**: Access Wikipedia, Wikidata, OpenLibrary, WolframAlpha, NASA APIs, and more.  
- **Real-Time Answers**: Quick calculations, data queries, and reference lookups.  
- **Topic Aggregation**: Collect results from multiple APIs for research projects in one unified view.  
- **Smart Query Suggestions**: Learn from previous searches to suggest related topics or content.

### Notes & Study Tools

- **Personal Notes**: Create, edit, and organize notes alongside your search results.  
- **Bookmarks & Collections**: Save articles, data points, and references for projects or exams.  
- **Research Workspaces**: Group notes and searches into topics, subjects, or projects.  
- **Highlight & Annotate**: Mark key points directly in retrieved content for fast review.

### Educational Utilities

- **Math Solver**: Solve equations, plot graphs, or calculate formulas via WolframAlpha or MathJS API.  
- **Science Visualizations**: Access NASA imagery, charts, and scientific data for projects.  
- **Dictionary & Translation**: Improve language skills using Dictionary API and translation tools.  
- **Task & Time Management**: Organize assignments and deadlines using Todoist API or internal planner.  
- **Automated Summaries**: Use AI-powered summaries (OpenAI API) for articles, papers, or research notes.

### Customization & Extensibility

- **API-Friendly Architecture**: Developers can integrate additional APIs or services.  
- **Open Source**: Modify features, add new tools, or improve existing modules.  
- **Student Plugins**: Add scripts or functions tailored for educational purposes.

## Supported APIs

| API Name           | Purpose                                    | Example Use Case                                      |
|-------------------|--------------------------------------------|------------------------------------------------------|
| Wikipedia API      | Encyclopedia articles                       | Search for historical events or scientific topics    |
| Wikidata API       | Structured knowledge from Wikipedia        | Retrieve data for charts, timelines, or research     |
| OpenLibrary API    | Books and authors                          | Find books for research papers or assignments        |
| Google Books API   | Book metadata and previews                 | Check book summaries, author info, and publication   |
| WolframAlpha API   | Math and scientific computations           | Solve equations, generate graphs, scientific data    |
| NASA APIs          | Astronomy and science resources            | Access satellite images or space research data       |
| OpenWeatherMap API | Weather data                               | Study climate patterns for geography or science      |
| MathJS API         | Math expression evaluation                 | Calculate formulas or check homework solutions       |
| Notion API         | Notes and project management               | Organize study notes, research projects, and ideas  |
| Evernote API       | Note storage and retrieval                  | Save lecture notes or research references            |
| Todoist API        | Task management                             | Keep track of study schedule and deadlines          |
| NewsAPI            | Current events and news articles           | Research news for assignments or reports            |
| Dictionary API     | Definitions, synonyms, translations        | Learn vocabulary and language studies               |
| OpenAI API         | Summaries, explanations, Q&A               | Generate study guides or answer questions           |

## Usage Guide

1. **Open crescent** in your browser.  
2. **Enter a query** in the search bar to retrieve results from multiple sources.  
3. **Create notes** and annotate key points directly within the workspace.  
4. **Organize projects** by subject, topic, or exam.  
5. **Use educational tools** such as calculators, graph generators, or AI summaries.  
6. **Save and export research projects** for review or sharing with peers.
