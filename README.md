# CrescentSearch
## A Search Engine for Students and Learners
**Version 1.0, 2025**

---

## Overview

CrescentSearch is a privacy-friendly search engine designed for students, researchers, and learners. It can access external APIs such as the **Wikipedia API**, while also providing a personal ecosystem for taking notes, organizing research, and managing study material. Everything is built to make studying and research easier, safer, and fully integrated.

---

## Key Features

### 1. API Access
- Retrieve information from public APIs like Wikipedia in real-time.
- Always respects API terms, rate limits, and usage policies.

### 2. Personal Study Ecosystem
- Take notes, bookmark resources, and organize research within the search engine.
- Keep all study material private, with no external tracking.

### 3. Learning-Friendly
- No ads, no unnecessary distractions.
- Safe for students under 18; perfect for educational and personal projects.

### 4. Customization
- Open for modification and improvement under the **CrescentDB Supreme Protection License**.
- Add new APIs or extend search functionality to suit your study needs.
